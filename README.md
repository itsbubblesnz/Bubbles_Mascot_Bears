# Mascot Bear Widget

A browser-source widget for streamers featuring adorable Pancake People that react to Twitch chat activity from specific assigned users.

## Features

* Custom assigned Mascot Bears (x9)
* Twitch chat interaction
* Streamer.bot integration
* Browser source compatible
* OBS compatible
* Streamlabs compatible
* Meld compatible
* Easy username configuration

## Requirements

* Streamer.bot
* Twitch account connected to Streamer.bot
* WebSocket Server enabled (127.0.0.1:8080)
* OBS Studio, Streamlabs Desktop, Meld Studio, Prism Live Studio, or similar software supporting Browser Sources

## Configuration

User assignments can be edited in:

usernames.js

Only usernames listed in the configuration file will be assigned a Pancake Person.

## Support

Please review the installation guides before requesting support.

Most setup issues are caused by:

* Streamer.bot not running
* WebSocket Server not enabled
* Incorrect usernames
* Browser Source not pointing to index.html
